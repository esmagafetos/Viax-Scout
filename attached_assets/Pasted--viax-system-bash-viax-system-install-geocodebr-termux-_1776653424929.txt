     docker run -p 8002:8002 -v geocodebr-cache:/root/.cache viax-geocodebr
     Configure GEOCODEBR_URL=http://IP-DO-SERVIDOR:8002 no .env

     Acesse Configurações → Instâncias → Padrão Gratuito
